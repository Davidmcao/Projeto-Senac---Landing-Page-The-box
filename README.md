# Site Responsivo - Multi-Seções

Este projeto é um site responsivo com múltiplas seções, desenvolvido para praticar organização de arquivos CSS em componentes, seções e layouts. Utiliza o Vite para facilitar o desenvolvimento frontend.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 modularizado
- Vite + Node.js

⚠️ **Importante:** este projeto foi configurado para funcionar com Vite (Node.js). Executá-lo com o Live Server pode causar problemas de rota para os arquivos CSS e JS.

## 📁 Estrutura

- `components/`: botões e formulários
- `layout/`: cabeçalho e rodapé
- `sections/`: áreas como "sobre", "projetos", "serviços", etc.

## 🚀 Como executar

```bash
npm install
npm run dev
